# 小遥搜索 - 模型选型

## 核心功能模型

| 用途 | 模型 | 说明 |
|------|------|------|
| 文本嵌入 | BAAI/bge-m3 | 中文文本向量化，支持多种语言 |
| 大语言模型 | qwen2.5:1.5b | 通过Ollama部署，轻量级本地LLM |
| 语音识别 | FasterWhisper base | 语音转文本，支持中英文 |
| 图像理解 | OFA-Sys/chinese-clip-vit-base-patch16 | 中文图像特征提取和相似度计算 |

```
                    │

```

## 部署方式

所有模型采用本地部署，确保数据隐私和离线使用：

- **BGE-M3**: 通过SentenceTransformers加载
- **Qwen2.5**: 通过Ollama容器化部署
- **FasterWhisper**: 本地模型文件
- **Chinese-CLIP**: 通过transformers库加载
