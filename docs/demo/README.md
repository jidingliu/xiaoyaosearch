# 小遥搜索高保真原型

基于 Vue3 + Vite + Ant Design 构建的多模态AI智能搜索应用原型。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📁 项目结构

```
docs/demo/
├── index.html              # 统一展示页面
├── search.html             # 搜索页面
├── settings.html           # 设置页面
├── help.html              # 帮助与关于页面
├── src/                   # 源代码
│   ├── search.ts          # 搜索页面入口
│   ├── settings.ts        # 设置页面入口
│   ├── index.ts           # 索引管理页面入口
│   ├── help.ts            # 帮助页面入口
│   ├── SearchApp.vue      # 搜索页面组件
│   ├── SettingsApp.vue    # 设置页面组件
│   ├── IndexApp.vue       # 索引管理页面组件
│   └── HelpApp.vue        # 帮助页面组件
├── package.json           # 项目配置
├── vite.config.ts         # Vite配置
├── tsconfig.json          # TypeScript配置
└── README.md             # 项目说明
```

## 📄 页面说明

### 1. 搜索主页面 (`search.html`)
- **多模态输入区**：支持语音、文本、图片三种输入方式
- **搜索设置**：显示当前AI模型和搜索范围
- **结果展示**：按相关度排序的搜索结果，支持预览和操作
- **状态栏**：显示模型状态、索引文件数量和搜索历史

### 2. 设置页面 (`settings.html`)
- **LLM模型设置**：云端API和本地Ollama配置
- **语音识别设置**：云端API和本地FastWhisper配置
- **视觉模型设置**：云端API和本地视觉模型配置
- **通用设置**：主题、语言、性能配置
- **隐私设置**：数据保护和使用统计设置

### 3. 索引管理页面
- **文件夹管理**：添加、删除、重建索引
- **索引统计**：文件数量、大小、格式统计
- **进度监控**：实时的索引进度显示
- **批量操作**：全部重建、清空等操作

### 4. 帮助与关于页面 (`help.html`)
- **使用教程**：详细的步骤指导
- **常见问题**：FAQ和解决方案
- **故障排除**：常见技术问题排查
- **联系支持**：多种联系方式和反馈渠道
- **关于信息**：版本信息和技术栈

## 🛠️ 技术实现

### 核心技术栈
- **Vue 3**：使用Composition API构建响应式界面
- **TypeScript**：类型安全的开发体验
- **Ant Design Vue 4.x**：企业级UI组件库
- **Vite**：快速的开发构建工具

### 组件设计
- **单文件组件**：每个页面独立设计，便于维护
- **响应式数据**：使用Vue 3的响应式系统管理状态
- **模块化结构**：清晰的文件组织和代码分离

### 交互设计
- **状态管理**：使用reactive管理组件状态
- **事件处理**：完整的用户交互响应
- **异步操作**：模拟真实的网络请求和数据处理

## 🔧 开发说明

### 本地运行
1. 安装Node.js环境（推荐v18+）
2. 克隆项目到本地
3. 运行 `npm install` 安装依赖
4. 运行 `npm run dev` 启动开发服务器
5. 访问 `http://localhost:3000` 查看原型

### 自定义配置
- **主题颜色**：修改 `a-config-provider` 的 `theme.token.colorPrimary`
- **布局样式**：调整各组件的CSS样式
- **交互逻辑**：修改Vue组件的事件处理函数

### 构建部署
```bash
npm run build
```
构建后的文件在 `dist` 目录中，可以部署到任何静态文件服务器。
