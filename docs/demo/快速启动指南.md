## 🎯 一键启动

### Windows 用户
```bash
# 双击运行
start.bat
```

### 手动启动
```bash
# 1. 进入项目目录
cd docs/demo

# 2. 安装依赖（首次运行）
npm install

# 3. 启动开发服务器
npm run dev
```

## 🌐 访问地址

启动成功后，您可以访问以下页面：

| 页面 | 地址 | 说明 |
|------|------|------|
| 🎯 **导航中心** | http://localhost:3000/navigation.html | **推荐入口**，所有页面导航 |
| 🔍 **搜索页面** | http://localhost:3000/search.html | 多模态搜索功能 |
| ⚙️ **设置页面** | http://localhost:3000/settings.html | 配置AI模型 |
| 📁 **索引管理** | http://localhost:3000/index-manage.html | 文件夹索引管理 |
| ❓ **帮助页面** | http://localhost:3000/help.html | 使用教程和FAQ |
| 🎯 **统一展示** | http://localhost:3000/index.html | iframe嵌入式展示 |

### 💡 推荐使用方式

1. **从导航中心开始**：访问 `navigation.html`，点击各个页面卡片在新窗口中打开
2. **独立页面体验**：每个页面独立打开，获得完整的用户体验
3. **对比查看**：使用统一展示页进行功能对比

- **Vue 3** + **TypeScript** - 现代化前端框架

如果遇到问题，请检查：

2. **网络连接**：某些功能需要网络
3. **端口占用**：默认使用3000端口
