# 小遥搜索 XiaoyaoSearch 开发排期表

## 项目概览

- **项目名称**: 小遥搜索 XiaoyaoSearch MVP版本
- **项目周期**: 10周（2024年12月 - 2025年2月）
- **团队规模**: 6人（前端2人、后端2人、测试1人、DevOps 0.5人、产品经理0.5人）
- **技术栈**: Electron + Vue3 + TypeScript + Python FastAPI + AI模型
- **目标**: 打造支持多模态AI智能搜索的本地桌面应用
