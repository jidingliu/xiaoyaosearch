智能文件搜索，重新定义知识管理

小遥搜索是一个跨平台的本地文件智能搜索工具，通过AI多模态理解能力，为个人用户提供文本、语音、图片、视频等多种文件的语义化检索服务

核心功能特性

AI智能搜索

基于深度学习的语义理解，支持自然语言查询和智能建议

多模态输入

支持文本、语音、图片多种搜索方式，满足不同使用场景

隐私保护

本地AI模型优先，确保用户数据完全私密安全

跨平台支持

Windows、macOS、Linux全平台支持，统一的使用体验
