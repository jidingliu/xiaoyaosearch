
我需要设计一个本地AI搜索的应用叫做“小遥搜索 XiaoyaoSearch”，他是一个本地应用（windows、mac），

主流程为：用户可以输入 语音（30秒内）、音频（30秒内）、文本、图片（png、jpg），
然后再通过AI大模型及多模态的能力转换成搜索的语义，再搜索本地电脑中文件
（视频（mp4、avi）、音频（mp3、wav）、文件（txt、markdown文件）、文档（office文件：excel、ppt、pdf））
的文件名和文件内容，从而达到精确找到文件及内容的目的，找到文件后，可以对文件进行预览、筛选。

核心功能有：
搜索主页面、
设置页面（
对大模型进行设置（可以使用云端的大模型api接口（设置api地址+访问令牌+LLM模型名称），也可以使用本地的ollama模型（设置api地址+模型名称），二选一）、
对语音识别模型进行设置（可以设置云端的api接口（设置api地址+访问令牌），也可以选择使用本地的fastwhisper模型（无需任何设置），二选一）、
对视觉模型进行设置（可以设置云端的api接口（设置api地址+访问令牌+视觉模型名称），也可以使用本地的视觉模型（无需任何设置），二选一）
）、
索引管理页面（
可以选择哪些文件夹进行索引建立、重建索引、删除索引等
）


页面要求有：现代化的搜索体验、使用本地搜索像使用谷歌搜索一样丝滑
