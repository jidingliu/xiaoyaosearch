# 小遥搜索 XiaoyaoSearch v2.0 - 高保真原型

## 🎯 项目概述

这是一个功能完整的**高保真原型项目**，基于产品需求文档和API规范，使用Vue3 + TypeScript + Ant Design Vue构建。项目实现了小遥搜索产品的核心功能，包括多模态AI搜索、文件索引管理、系统设置等完整功能模块。

### ✨ 完成的核心功能

#### 1. 智能搜索界面
- **多模态输入**：文本、语音、图片三种输入方式切换
- **语音录制**：实时波形动画，30秒时长限制，语音转文字模拟
- **图片上传**：拖拽上传，图片预览，AI内容分析模拟
- **搜索选项**：高级搜索设置，文件类型过滤，相似度调节
- **实时搜索**：搜索状态反馈，结果统计，加载动画

#### 2. 搜索结果展示
- **精美卡片设计**：现代化UI，悬停效果，收藏状态
- **智能匹配度显示**：圆形进度条，颜色编码，百分比显示
- **内容预览**：关键词高亮，文件信息展示
- **快速操作**：预览、打开、收藏、复制路径等快捷功能
- **文件类型识别**：文档、音频、视频、图片等类型图标

#### 3. 索引管理系统
- **任务监控**：实时进度显示，状态管理
- **文件夹管理**：添加、删除、重建索引
- **统计仪表板**：文件数量、索引大小、成功率统计
- **详细日志**：实时处理日志，错误信息展示
- **批量操作**：多选、批量删除等功能

#### 4. 系统设置页面
- **AI模型配置**：云端API和本地Ollama配置
- **语音识别设置**：FastWhisper本地模型配置
- **搜索参数设置**：默认结果数、相似度阈值
- **界面主题设置**：浅色/深色主题，语言选择
- **数据管理**：数据目录选择，历史记录管理

#### 5. 帮助和文档系统
- **快速入门指南**：3步开始使用流程
- **详细使用教程**：语音搜索、图片搜索等操作说明
- **常见问题解答**：FAQ和故障排除指南
- **项目关于信息**：版本信息、技术栈、开源许可

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 启动方式

#### 方式一：使用启动脚本（推荐）
```bash
# 双击运行
start.bat
```

#### 方式二：手动启动
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 访问地址
- 开发环境：http://localhost:5173
- API文档：http://127.0.0.1:8000/docs（后端服务启动后）
