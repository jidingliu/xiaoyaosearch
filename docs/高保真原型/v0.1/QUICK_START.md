# 🚀 小遥搜索 v0.1 高保真原型 - 快速启动指南

## 📋 项目概述

小遥搜索 v0.1 高保真原型是一个基于 Vue 3 + TypeScript + Ant Design Vue + Vite 构建的现代化多模态AI智能搜索应用原型。

### ✨ 核心特色

- **🎨 独特视觉设计** - 深邃科技蓝紫渐变主题，斜切几何搜索框
- **🎭 多模态输入** - 支持语音、文本、图片三种搜索方式
- **💫 丰富动画效果** - 流体式过渡动画，悬浮状态指示器
- **📱 完美响应式** - 适配桌面、平板、移动设备
- **🎯 交互丰富** - 语音波形动画，搜索建议，骨架屏加载

## 🛠️ 环境要求

- **Node.js**: 18.0.0 或更高版本
- **npm**: 8.0.0 或更高版本（或 yarn/pnpm）

## 🐛 常见问题

### Q: 启动时端口被占用怎么办？
A: 修改 `vite.config.ts` 中的 `server.port` 配置，或使用 `npm run dev -- --port 3001`

### Q: 样式显示不正常？
A: 确保使用现代浏览器，清除浏览器缓存，检查SCSS编译是否正常

### Q: 搜索功能没有响应？
A: 这是原型版本，搜索功能使用模拟数据，主要用于展示交互效果

### Q: 在移动设备上体验不佳？
A: 建议在桌面端体验完整功能，移动端主要用于响应式测试
