# 小遥搜索 v0.1 高保真原型

## 项目概述

小遥搜索是一款支持多模态AI智能搜索的本地桌面应用高保真原型，基于 Vue 3 + TypeScript + Ant Design Vue + Vite 构建。

## 🎨 设计特色

### 独特视觉风格
- **深邃科技蓝紫渐变主题** - 营造沉浸式搜索体验
- **斜切几何搜索框** - 独特的视觉识别度
- **悬浮多模态指示器** - 直观的输入模式状态显示
- **动态光影系统** - 多层次发光效果和装饰元素

### 核心功能
- **多模态输入** - 支持语音、文本、图片三种搜索方式
- **智能搜索结果** - 卡片式展示，包含文件预览和操作按钮
- **实时状态反馈** - 录音波形、搜索进度、匹配度显示
- **响应式设计** - 完美适配桌面、平板、移动设备

## 🎯 核心组件

### MultiModalIndicator 多模态指示器
三个悬浮圆圈显示当前输入模式状态：
- 🔊 语音模式 - 录音时显示波形动画
- 📝 文本模式 - 默认输入模式
- 🖼️ 图片模式 - 支持图片上传搜索

### FloatingSearchBox 悬浮搜索框
- **斜切几何设计** - 独特的视觉效果
- **动态边框** - hover 和 focus 时的发光效果
- **实时建议** - 搜索历史和智能建议
- **录音计时器** - 30秒倒计时和进度条

### ResultCard 搜索结果卡片
- **文件信息展示** - 名称、路径、大小、类型
- **匹配度显示** - 相关性分数和高亮文本
- **快捷操作** - 预览、打开、收藏、删除
- **骨架屏** - 加载状态的优雅占位

## 🎨 设计系统

### 色彩体系
- **主色调**: 深邃科技蓝紫渐变 (\`#1A1F4B → #4A148C → #00BCD4\`)
- **强对比色**: 青色 (\`#00E5FF\`)、洋红 (\`#FF006E\`)、金色 (\`#FFD700\`)
- **深色主题**: 沉浸式背景，减少视觉疲劳

### 动画效果
- **页面切换**: 流体式过渡动画
- **悬浮效果**: hover 时的抬升和发光
- **录音波形**: 实时音频可视化
- **加载动画**: 智能脉冲和进度指示

### 响应式布局
- **桌面 (1024px+)**: 三列卡片网格
- **平板 (768px-1023px)**: 双列布局
- **移动 (480px-767px)**: 单列垂直布局
- **小屏 (480px-)**: 紧凑模式

## 🎯 特色功能

### 语音搜索
- **实时波形显示**: 5个动态波形条
- **30秒录音限制**: 自动停止和倒计时
- **语音转文字**: AI 模型识别搜索意图
- **录音质量控制**: 自动降噪处理

### 图片搜索
- **拖拽上传**: 直接拖拽图片到搜索框
- **格式支持**: JPG、PNG、WebP 等主流格式
- **AI 视觉理解**: 提取图片内容和概念
- **相似图片搜索**: 基于视觉特征匹配

### 智能建议
- **搜索历史**: 最近搜索记录
- **智能补全**: 基于输入的实时建议
- **文件建议**: 根据内容推荐相关文件
- **快捷选择**: 键盘导航快速选择

## 🔄 API 集成

### 搜索服务
- \`POST /api/search\` - 文本搜索
- \`POST /api/search/multimodal\` - 多模态搜索
- \`GET /api/search/history\` - 搜索历史
- \`GET /api/search/suggestions\` - 搜索建议

### 索引管理
- \`POST /api/index/create\` - 创建索引
- \`GET /api/index/status/{id}\` - 索引状态
- \`DELETE /api/index/{id}\` - 删除索引
- \`GET /api/index/list\` - 索引列表

### WebSocket 实时通信
- \`/ws/index/{id}\` - 索引进度推送
- \`/ws/search-suggest\` - 搜索建议推送
