# 小遥搜索 XiaoyaoSearch MVP - 高保真原型

> 一款支持多模态AI智能搜索的本地桌面应用MVP高保真原型

## 🎯 项目概述

这是小遥搜索产品的MVP版本高保真原型，基于Vue3 + TypeScript + Ant Design Vue构建，专注于核心功能的完整实现和优秀的用户体验。

### ✨ 核心特性

- 🎯 **多模态搜索**：支持文本、语音、图片三种输入方式
- 🔍 **智能搜索**：AI驱动的语义理解和内容匹配
- 📱 **现代UI**：基于Ant Design Vue的专业界面设计
- ⚡ **实时交互**：完整的用户交互流程和状态反馈
- 🎨 **响应式设计**：完美适配桌面和移动设备
- 🔧 **Mock数据**：完整的模拟API和测试数据

### 🚀 MVP版本亮点

- ✅ **简洁设计**：去除复杂功能，专注核心搜索体验
- ✅ **大尺寸交互**：优化的多模态切换按钮，易于点击
- ✅ **品牌强化**：清晰的"小遥搜索"品牌标识
- ✅ **MVP导向**：只包含最核心的搜索和管理功能
- ✅ **用户友好**：直观的操作流程和清晰的信息展示

## 🛠️ 技术栈

- **前端框架**：Vue 3 + Composition API
- **类型系统**：TypeScript 5.0+
- **UI组件库**：Ant Design Vue 4.x
- **构建工具**：Vite 5.x
- **状态管理**：Pinia
- **路由管理**：Vue Router 4
- **HTTP客户端**：Axios
- **样式方案**：CSS Variables + Scoped CSS

## 📦 安装和运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0 或 yarn >= 1.22.0

### 安装依赖

```bash
# 克隆项目
git clone <repository-url>
cd xiaoyaosearch-prototype

# 安装依赖
npm install
```

### 开发环境

```bash
# 启动开发服务器
npm run dev

# 访问应用
# 浏览器打开: http://localhost:5173
```

### 构建生产版本

```bash
# 构建项目
npm run build

# 预览构建结果
npm run preview
```

## 🔧 配置说明

### 环境变量

创建 `.env` 文件并配置以下变量：

```env
# API配置
VITE_API_BASE_URL=http://127.0.0.1:8000

# 应用配置
VITE_APP_TITLE=小遥搜索 XiaoyaoSearch MVP
VITE_APP_VERSION=1.0.0
VITE_APP_SLOGAN=多模态智能搜索，让文件触手可及

# 功能开关
VITE_ENABLE_MOCK=true
VITE_ENABLE_DARK_MODE=false
```

### Mock数据

项目内置了完整的Mock数据服务，包含：

- **搜索结果模拟**：17个真实测试文件的完整数据集
- **多格式支持**：文档、音频、视频、图片文件类型
- **索引进度模拟**：实时状态更新和进度显示
- **系统配置模拟**：语音设置和搜索参数
- **用户交互反馈**：完整的操作提示和状态消息

### 数据集说明

Mock数据基于真实测试目录，包含：
- **文档文件**：PDF、DOCX、PPTX、XLSX、MD、TXT
- **音频文件**：MP3、WAV
- **视频文件**：MP4、AVI
- **图片文件**：JPG、JPEG、PNG

## 🚀 开发指南

### 添加新页面

1. 在 `src/views/` 创建页面组件
2. 在 `src/router/index.ts` 添加路由
3. 更新导航菜单

### 添加新组件

1. 在 `src/components/` 创建组件
2. 遵循TypeScript类型规范
3. 使用Composition API编写逻辑
4. 添加Scoped样式

### API集成

1. 在 `src/api/` 添加服务文件
2. 定义TypeScript接口
3. 实现Mock数据
4. 处理错误和加载状态
